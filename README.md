# Bernice — Portfolio

A folder-shaped portfolio site for Bernice Toh Jiayi. Dark pink tab toolbar + light pink content face; every page renders inside the folder.

## Stack

Vite · React 18 · react-router-dom (one lazy-loaded JSX per page)

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build   # outputs to dist/
```

## Deploy

Deployed on Vercel (SPA rewrites configured in `vercel.json`).

See `PROJECT_GUIDE.md` for the full structure, route map, component registry, and theme reference.
